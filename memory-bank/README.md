# Memory Bank

This directory stores persistent project context for Cline sessions in this repository.

## Files
- `memory-bank/projectbrief.md`: project mission, scope, and repository layout.
- `memory-bank/productContext.md`: operator workflows, expectations, and success criteria.
- `memory-bank/systemPatterns.md`: architecture and idempotency patterns used in setup automation.
- `memory-bank/techContext.md`: technologies, tools, dependencies, and platform constraints.
- `memory-bank/activeContext.md`: current snapshot, open decisions, and immediate next actions.
- `memory-bank/progress.md`: completed work, ongoing tasks, backlog, and change history.

## Update rules
- Update `activeContext.md` and `progress.md` after meaningful repository changes.
- Keep entries concise, factual, and specific to exact file paths.
- Reflect macOS-first assumptions and idempotency behavior from `setup.sh`.
